📌 Project Title

Project Management Software System

📖 Brief Description

The Project Management Software System is a web-based application developed to help teams and organizations efficiently manage projects, tasks, workflows, and collaboration activities.

The system allows users to create and manage projects, assign tasks, track progress, manage workflows using Kanban boards, monitor analytics, and collaborate with team members in real time. The software improves productivity, communication, and project organization by providing a centralized platform for project tracking and management.

This project is developed using modern full-stack web technologies with responsive UI/UX design and scalable backend architecture.

⚙ Installation / Execution Steps


1️⃣ Clone the Repository
git clone https://github.com/your-username/project-management-software.git


2️⃣ Navigate to Project Folder
cd project-management-software


🚀 Frontend Setup
Install Frontend Dependencies
npm install
Run Frontend Development Server
npm run dev

Frontend will run on:

http://localhost:3000
🚀 Backend Setup
Navigate to Backend Folder
cd backend
Install Backend Dependencies
npm install
Create .env File
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
Run Backend Server
npm run dev

Backend will run on:

http://localhost:5000
🗄 Database Setup
PostgreSQL / Supabase Configuration
Create a PostgreSQL or Supabase database.
Execute all SQL schema files.
Configure database credentials in the .env file.
Start backend server.
